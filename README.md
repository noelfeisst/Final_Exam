# Web Scraping, APIs, and Regressions
MSc Final exam tasked with utilizing web scraping and APIs to gather data for text analysis and regressions. As this was an exam there are written explanations of the data, the meaning of regression varaibles, and the outcomes of the regressions. 

The Wikipedia API was used to gather data about the winners of Ru Paul's Drage Race. Text analysis was performed where code was written to discover insights such as most popular word or word count. Further details can be found in the code notebook.
